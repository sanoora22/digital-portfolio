Digital Portfolio

A personal portfolio website built with HTML and CSS to showcase my background, skills, and projects, with a downloadable resume.

Overview

This is a single-page personal portfolio site with smooth-scrolling navigation between sections: About Me, Education, Skills, Projects, and Resume. It's designed as a simple, clean way to present my profile to recruiters and visitors.

Features


Responsive single-page layout with a fixed header and profile picture
Smooth-scroll navigation between sections
About Me section summarizing my background and career interests
Education, Skills, and Projects sections
One-click resume download (PDF) from the Resume section


Tech Stack


HTML — page structure and content
CSS — styling, layout, and visual design (embedded in index.html)
JavaScript — smooth-scroll behavior for in-page navigation


Project Structure

digital-portfolio/
├── index.html     # Main page: structure, styling, and navigation script
├── photo.jpg       # Profile picture
├── resume.pdf      # Downloadable resume
└── README.md

Getting Started

No build steps or dependencies — this is a static site.

bashgit clone https://github.com/sanoora22/digital-portfolio.git
cd digital-portfolio

Then simply open index.html in a browser, or serve it with any static file server.
